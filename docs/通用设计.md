# 通用设计

适用于 [cf-kv-filesystem](https://github.com/joyanhui/cf-kv-filesystem) 和 [cf-shorturl](https://github.com/joyanhui/cf-shorturl)。

**技术栈:** Bun + React + React Router + Hono + OpenAPI + Tailwind CSS + shadcn/ui + Zod，D1 数据库用 Drizzle ORM。

---

## 目录结构

```
src/

├── render.tsx            # React SSR 辅助
├── jwt.ts                # JWT 原语（HMAC-SHA256）
├── lib/                  # 工具/业务逻辑
│   ├── auth.ts           # 管理员认证（密码 + JWT）
│   └── types.ts          # 数据模型
├── admin/                # 管理后台 API（Hono 子应用）
│   ├── index.ts          # 路由注册
│   ├── auth.ts           # JWT cookie 校验
│   └── api_*.ts          # 各功能模块
├── db/                   # 数据库层（仅 cf-kv-filesystem）
│   ├── index.ts          # getDB 工厂
│   └── schema.ts         # Drizzle 表定义
├── frontend/             # SSR + SPA
│   ├── App.tsx           # BrowserRouter
```

## 路由 & 认证

ADMIN_PATH 环境变量配置后台路径，默认 `/admin`，禁止为 `/`。入口通过 pathname 前缀判断分发到 admin 子应用或公开 app，子应用内部路径不包含 ADMIN_PATH 前缀。

JWT 签名用 JWT_ADMIN_SECRET 环境变量（Secret 类型），HttpOnly Cookie 24h。admin 路由通过 `requireAuth` 中间件保护。

## 前端

所有前端代码在 `src/frontend/`，SPA 通过 BrowserRouter 路由，basename 运行时从 `window.location.pathname` 提取。未认证重定向到 `/login`，认证后显示 Dashboard。i18n 通过 Accept-Language + locale cookie（SSR）或 localStorage + navigator.language（SPA）判定，LANG_SCRIPT 同步两者。

首页和错误页通过 `renderToStaticMarkup` 服务端渲染，无客户端 bundle。首页写入 `caches.default`（TTL 1h），所有文案支持中英文。

### 设计要求
1. **首页不得包含 admin panel 链接** — 首页是公开页面，仅显示 GitHub 链接
2. **所有页面必须有双语言切换** — SSR 页面（首页/错误页）通过 Layout 注入的浮动按钮切换；SPA（管理面板）通过 Dashboard 和 LoginPage 中的 toggle 按钮切换
3. **退出登录必须清除会话** — 管理面板退出按钮调用 `DELETE /api/login` 清除 JWT Cookie，前端同步清除 `authed` 状态，跳转到登录页

## Zod + OpenAPI

Schemas 集中在 `src/schemas/index.ts`，API handler 通过 `schema.parse/safeParse` 校验替代 `as any`。公开 API 使用 OpenAPIHono，路由通过 `createRoute` 声明，自动生成 `/openapi.json`。

## 数据存储

cf-kv-filesystem 用 D1 + Drizzle ORM，函数签名保持 `(d1: D1Database, ...)`，内部调用 `getDB()`。cf-shorturl 通过 Service Binding HTTP 调用 kv-filesystem Worker（`src/lib/kv-fs.ts` 封装），数据以 JSON 存取，pkey 以 `shorturl_` 前缀。

## 构建

`bun scripts/build-css.ts` 编译 Tailwind CSS + esbuild 打包 SPA，esbuild alias `@` → `src`。shadcn 组件用 `npx shadcn@latest add` 添加到 `src/frontend/components/ui/`。构建产物（CSS + JS）内联到 `admin.gen.ts`。

## 环境变量

| 变量 | 类型 | 适用 |
|------|------|------|
| ADMIN_PATH | Plain text | 通用（默认 /admin） |
| JWT_ADMIN_SECRET | Secret | 通用 |
| KV_FS_API_KEY | Secret | cf-shorturl |
| JWT_ITEM_SECRET | Secret | cf-kv-filesystem |

## 项目差异

| 维度 | cf-kv-filesystem | cf-shorturl |
|------|-----------------|-------------|
| 存储 | D1 + R2 + Drizzle ORM | HTTP Client → kv-filesystem Worker |
| 数据库表 | kv_items, config, api_keys | 无 |
| 额外 API | JWT 分享、缓存清除、R2 治理 | Turnstile 验证码、Basic Auth |
| 定时任务 | Cron 每日 6:00 清理 | 无 |
| 特有文件 | cleanup.ts, api_meta.ts, api_apikey.ts, config.ts, api_format.ts, api_cache.ts | lib/kv-fs.ts, lib/types.ts |
